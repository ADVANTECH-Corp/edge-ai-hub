# Edge AI Hub

Edge AI Hub is a developer-focused repository of practical Edge AI application examples for Advantech edge computing platforms.

It provides scenario-based AI cases across domains such as Vision AI, Generative AI, Vision-Language Models, robotics, industrial monitoring, smart retail, medical AI, and smart manufacturing.  
Each case includes a structured README with deployment steps, runtime requirements, execution commands, and expected results, helping developers quickly evaluate and deploy AI applications on supported Advantech devices.


---

## Why Edge AI Hub?

Building Edge AI applications usually requires developers to handle many complex steps:

- Selecting the right AI model
- Preparing the runtime environment
- Installing vendor SDKs
- Converting or optimizing models
- Running inference on edge devices
- Validating performance and deployment results

Edge AI Hub simplifies this process by providing structured, scenario-based AI examples with step-by-step deployment guides.

Developers can follow the SOP, deploy the application to supported Advantech Edge AI devices, and verify the result in a real environment.

---

## What You Can Find in This Repository

Each AI case provides:

- Application introduction
- Supported hardware platform
- Runtime environment
- Model and framework information
- Docker-based deployment guide
- Step-by-step SOP
- Demo execution command
- Expected output
